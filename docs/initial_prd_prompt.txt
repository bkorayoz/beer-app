Tech Stack Requirements

UI Library: shadcn/ui (Radix Primitives).

This project is a showcase for AI Search Optimization. Code structure is as important as visual fidelity.

Semantic HTML: You must use <header>, <main>, <article>, <section>, <aside>, <nav>, and <footer> tags correctly. NEVER use a <div> where a semantic tag is appropriate.

Structured Data (JSON-LD): Every page must include a rigid "AEO Layer" that injects schema.org data.
